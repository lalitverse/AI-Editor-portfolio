# 🎬 Premium Cinematic AI Artist Portfolio

A stunning, highly performant dark-themed portfolio crafted for Cinematic AI Artists, Digital Creators, and Video Editors. Built with modern web technologies to guarantee hyper-smooth interactions and a premium glassmorphic feel.

## 🚀 Built With
- **React.js** (Vite)
- **Tailwind CSS** (Utility-first styling & Custom Background Gradients)
- **Framer Motion** (Buttery smooth entrance animations, Spring physics, and 3D Interactive elements)
- **Lucide Icons**

## ✨ Key Features
1. **Interactive 3D Carousel Gallery:** A fully responsive, swipeable 3D ring of bespoke cinematic artworks that automatically scales its Z-radius mathematically based on the exact image count. Continuously auto-rotates for a dynamic showcase.
2. **Interactive Project Lightbox:** Click any bento grid project to seamlessly pop it out into a centered screen modal using Framer Motion's `layoutId` scaling transition. Features a premium backdrop blur.
3. **Contact Modal:** A beautiful glassmorphic layer allowing instant contact securely and gracefully, keeping primary social links immediately accessible.
4. **Magnetic Hover Buttons:** Primary action buttons implement physical spring dynamics, gently pulling towards the user's cursor for a futuristic, tactile interface depth.
5. **Responsive Glassmorphism:** Implements dark, translucent overlays and border treatments to create a "Neo-Tactile" 3D layered interface.

## 🛠️ Running Locally
1. Clone the repository
2. Run `npm install` to grab dependencies (`framer-motion`, `lucide-react`, `tailwindcss`)
3. Run `npm run dev` or `npm run build`
